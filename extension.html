MP1 Writeup

Overview and Usage

This chrome extension allows users to choose a new background color for whichever website of their choice. It contains 3 color pickers that have a preset theme to select from or users can choose to preset another theme of colors. This gives users the opportunity to set a theme for all the websites they visit!

Issue Deep Dive

For this mini project, I had to learn basically everything with no previous knowledge of using Javascript or the DOM. This was really difficult for me because it took a lot of time to understand what the examples have done and also what the capabilities are for Javascript. Because of this steep learning curve, I had a lot of logic issues. One of them being figuring out how to make the checkbox send a message to the content script. I learned how to use console.log to figure out what was and what wasn't working. Another thing I learned along the way was the syntax for writing functions and how to navigate the scope of certain variables. I also learned how to cross reference my variables and id names across content scripts, html, and javascript.
