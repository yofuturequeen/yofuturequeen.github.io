Overview and Usage

This game is called Strawberraroll and is the final version of my game. Play as Sanrio's Cinnamaroll and collect strawberries and flour to create a cake! Make sure to avoid the poison or your cake will be ruined. Strawberraroll gets increasingly harder as you continue to play so be extra careful. Play a game of Strawberraroll here!

Issue Deep-Dive

The biggest problem I ran into for finishing up this game was adding Firebase. I would say this was mainly logic error as I felt completely unfamiliar with using it and adding to my existing code. With Firebase, there are a lot of components that link together and I think it was most difficult for me to figure out what should in which file. What I mean by this is differentiating what needs to go in my html file vs js file and then learning how they should commmunicate with one another. In the end, I tried for a very long time to implement a leaderboard using peers and online work but it did not work out and would consistently crash my game. A large part of this is due to the fact that I still don't have the best understanding of javascript outside of making games. I will attempt to revisit this but I simply could not figure this out. You can take a look at my attempted code in my firebase.js file.
