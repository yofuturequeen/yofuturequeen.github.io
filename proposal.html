Final Project Proposal

Description

For my final project I will be revisiting my game miniproject Strawberraroll. This game will be updated in a variety of ways including incorporating a new library (p5 sound) by adding a soundtrack, adding animations, and making the game restart upon click.

Prototype vs Final

Prototype Version

The prototype version will be how I test out a lot of the new functionality I want to add. This version will not have all of the complete parts such as the strawberries and characters. However, it'll have core functionality like adding a sound effect when a sprite overlaps another. The prototype version will also be where I learn how to make animations such as making a cake using the ingredients collected at the end of the game.

Final Version

The final version of my game will have all the new functionality using the actual characters and game pieces. It will include pieces I design on my own such as the text for game over and high scores. The game will also take up user's entire screens which it does not currently. Lastly a piece I want to add to the game is making it progressively get harder as the user continues to play.

Development Process

  1. Familiarize myself with p5 sound and add basic sound effects to different sprites
  2. Create cake making/game ending animation in separate developer
  3. Create styled text to represent score and game over
  4. Finish prototyped version with above changes
  5. Start on final version by adjusting elements to make Strawberraroll full screen
  6. Make the game harder by adding if else statements to make sprites spawn more quicly
  7. Add sound and animation elements from prototyped version
  8. Make the game restart upon click rather than refreshing the page

Known Unknowns

  • How to use p5 sound
  • How to create and add animations in p5
  • How to make the game progressivley harder
  • How to create a restart function for the entire game

Stretch Goals

  • Add other characters that users can choose from
  • Add different outfits for the characters
  • Add a game won element if users get to a certain score/# of cakes made
